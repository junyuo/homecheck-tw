## 核心流程

- [ ] 首頁未宣稱尚未接入的公車、公園或市場能力。
- [ ] 空白查詢不能提交；切換縣市或行政區後必須重新確認地圖位置。
- [ ] 載入大安區範例後可提交，且結果與比較頁持續顯示範例模式。
- [ ] 結果首屏顯示四面向狀態、前三項優先問題與資料缺口。
- [ ] unavailable 或覆蓋不明來源顯示「資料不足」，不得顯示為 0 件、0 處或零風險。
- [ ] 年度成交少於 5 筆時顯示趨勢警示；事故顯示三年總數、年平均、涵蓋年度及非安全分數提示。
- [ ] 舊 Local Storage 快照可開啟；缺少年度樣本數時顯示「舊快照未保存」。
- [ ] v7 以前快照的市場欄顯示「舊快照，請重新查詢」，不得顯示 0 間。
- [ ] 市場 unavailable 時結果、比較與列印顯示「資料不足」；正式時顯示數量、最近市場與直線距離。
- [ ] 可加入並比較 3 間物件，表格包含四面向狀態、資料缺口與未完成查證數。
- [ ] 資料來源頁的正式／部分／不足狀態及臺灣日期與 manifest 一致。

## 裝置與可存取性

- [ ] Chrome 最新穩定版：1280px 桌面完整流程、鍵盤操作與焦點順序。
- [ ] Safari 最新穩定版：1280px 桌面完整流程、鍵盤操作與焦點順序。
- [ ] 375px 實機或等效 viewport：無水平溢出、表單與地圖可操作、首屏摘要可讀。
- [ ] VoiceOver：表單標籤、地圖說明、狀態與價格趨勢表格可理解。

## 列印／PDF

- [ ] Chrome 與 Safari 列印預覽均包含物件摘要、價格趨勢、四面向結論、資料日期與看屋清單。
- [ ] 操作按鈕、導覽列及互動地圖不出現在列印內容。
- [ ] A4 直向無重要內容遭裁切，標題不落在頁尾，表格列不跨頁破裂。
- [ ] PDF 文字可選取，繁體中文字型無缺字。
